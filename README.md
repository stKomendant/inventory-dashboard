# Inventory Dashboard

Admin dashboard built with React, TypeScript and Tailwind.

# Features

- Dashboard statistics
- Products CRUD
- Orders page
- Search & filtering
- Sorting & pagination
- Charts with Recharts
- Modal editing
- Delete confirmation
- Loading states
- Error handling
- Skeleton loading

# Tech Stack

- React
- TypeScript
- Tailwind CSS
- Recharts
- Vite
